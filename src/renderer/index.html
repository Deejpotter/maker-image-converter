Core Functionality

Standardizes images into 800Ã—800px WebP files (80% quality). The logic preserves aspect ratio by fitting images inside a pure white background..

Instructions

Simply select a folder containing your images and choose an operation mode then the photos in that folder will be processed according to the app rules.

Output Location

Process results are saved to a webp/ folder created inside the same directory as your source images.

Image Processing Logic

  • Primary Assets: Files ending in 01 (e.g., SKU-01.jpg) are processed without watermarks.
  • Secondaries: All other files receive a centered watermark. (e.g., SKU-02.jpg)
  • Dimensions: Files containing your Keyword (default: -dims) receive a diagonal watermark. (e.g., SKU-dims-07.jpg)

Application Modes

Full Run: The master workflow combining all conversion and heuristic branding.
Convert only: Pure resizing and WebP conversion. No watermarks applied.
Overlay only: Resizes to 800x800 and applies centered overlays to all non-primary files.
Diagonal only: Resizes and applies diagonal overlays to files matching the Dimensions keyword.
