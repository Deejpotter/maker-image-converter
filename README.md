# Maker Image Converter

A simple tool to batch convert images to WebP format with white background padding.

## Features
- Convert any image folder to WebP (800x800px, white background)
- Single executable distribution (.exe) for non-technical users
- Preserves original folder structure under `webp/` output directory

## Usage
1. Run the executable
2. Select an input folder containing images
3. The converted WebP files will appear in a `webp` subfolder

## Requirements
- Windows 10/11
- No additional installations required (standalone .exe)
